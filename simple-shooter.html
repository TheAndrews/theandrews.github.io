Simple Shooter Game

A simple shooter made with Unreal 5

This game contemplates 2 game modes, a deathmatch style where victory is achieved by killing all enemies and a capture the flag game mode where the team wins by dominating all flag areas.

Features:

  • First Person and Third Person Camera views
  • Animations handled by Animation-Graphs + Blend Spaces
  • Health Pool
  • Ammunition
  • Enemy AI built with Behaviour Trees + Blackboards, with custom Behaviour Tree services
    • On Kill'em All Game Mode Enemies wonder around searching for player to shoot
    • On Capture the Flag Game Mode Enemies go to flags that are not dominated to dominate, if player seen then shooting starts if player killed they go back to dominate flags
  • Character Respawn on Capture the Flag Game Mode

Most of the code was done only on C++ where Blueprints were used on ui elements and other more "configuration based" tasks.
